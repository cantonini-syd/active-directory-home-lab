# Active Directory Home Lab

A hands-on home lab project simulating a small enterprise environment using 
Windows Server 2022 and Windows 11 clients in VirtualBox. This lab covers 
AD DS setup, user/group management, Group Policy, DNS, DHCP, and common 
help desk tasks.

## Lab Environment
- **Hypervisor:** Oracle VirtualBox 7.1.4
- **Domain Controller:** Windows Server 2022 (Desktop Experience)
- **Client:** Windows 11
- **Host RAM allocated:** 8GB to server VM, 2 vCPUs

## Skills Demonstrated
- Virtualization with VirtualBox
- Windows Server 2022 installation and configuration
- Active Directory Domain Services (AD DS)
- DNS and DHCP configuration
- Group Policy Objects (GPO)
- User and OU management
- PowerShell and CMD administration
